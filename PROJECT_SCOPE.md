# Scope

Этот пакет нужен для анализа второй модели и визуализации выбора цели.

Разрешено:
- сравнение моделей;
- предсказания на изображениях;
- live-preview детектора;
- визуализация выбранной цели;
- создание демо-видео.

Не входит:
- управление CS2;
- управление мышью;
- управление клавиатурой;
- стрельба;
- онлайн-матчи;
- обход античита;
- чтение памяти игры;
- изменение файлов игры;
- инъекции;
- хуки.
