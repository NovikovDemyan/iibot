# Краткие правила разметки врагов

## Нормально

Коробка:
- охватывает видимую модель врага;
- может включать немного фона;
- может быть чуть больше силуэта;
- если враг частично скрыт, размечается только видимая часть.

## Плохо

Коробка:
- захватывает слишком много карты;
- размечает только голову, если виден весь человек;
- сильно обрезает тело без причины;
- включает другого врага;
- размечает союзника/оружие/труп/фон как врага.

## Практический критерий

Если человек по коробке понял бы: “да, здесь враг”, значит для первого обучения пойдёт.

Лучше иметь 200 умеренно нормальных кадров, чем 30 идеально размеченных.
